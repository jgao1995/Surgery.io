  • Device Database Management

      How do I add a device to the database?

      Click the Add Device link in the navigation bar. Select the Device Type you want to create a new device instance of, and click Next. Then, fill in the necessary fields for the Device. If the Manufacturer, Brand Name, or Description is unknown, please fill in "N/A". You can change the unit of any numerical field to centimeters (cm), inches (in), or French (Fr).

      How do I add a device type to the database?

      Click the Add Device Type link in the navigation bar. Enter the name of the Device Type (e.g. Wire). It will automatically have Manufacturer, Brand Name, and Description already as fields. Enter each of its additional field names that have numerical measurements (e.g. width, minimum inner diameter, etc.) You can add another field name by clicking the Add another field button. You can submit your new device type by clicking the blue Add button.

      How do I view details about a specific device in the database?

      Click the Search link in the navigation bar. Enter anything you know about the device, matching its product_type, manufacturer, brand, or description. You can click on any result that appears in the gray dropdown to view details about it.

      You can also add video links on a device's view page. In the input field under "Videos", enter a valid YouTube video link (for example: https://www.youtube.com/watch?v=OsOiTf2A36E), and click the Add video button.

      To add a new note, type any comment about a device into the input box and click "Add notes". Any new notes will be appended to the existing list of notes.
